### Features
- django 2.1.7
- Python3.6
- Linux
- Postgresql
- django-debug-toolbar
- bootstrap 4.3.x
- authentication
- logging configuration
- environment settings (+ github workflows CI)
- django-extensions
- rm_migrations
- ajax-csrf token
- tests


#### Usage
```shell

django-admin startapp --template=https://github.com/sifhic/django/archive/master.zip app1
```
