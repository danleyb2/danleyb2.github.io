Converting the PlateRecognizer Installer into a Docker Desktop Extension, enabling users to easily install and manage license plate recognition software within their Docker environment.

#### Key Features
- Install/management of PlateRecognizer software via Docker UI
- Configurable settings through extension panel
- Direct access to `guides.platerecognizer.com/docs/installer`
